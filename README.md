## Installation
1. Clone repository
```
git clone https://github.com/dylanvans/minor-ultimate && cd minor-ultimate
```
2. Install dependencies
```
npm install
```
3. Setup .env variables
```
DATABASE
SESSION_SECRET
SESSION_KEY
LEAGUEVINE_REDIRECT_URI
LEAGUEVINE_CLIENT_ID
LEAGUEVINE_CLIENT_SECRET_KEY
```
4. Start server
```
npm start
```
